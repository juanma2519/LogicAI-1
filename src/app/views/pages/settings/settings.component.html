Text → Image

1) Un modelo de texto redacta el prompt perfecto.

2) Un modelo de imagen genera la imagen final.

Image → Video

1) Modelo de imagen (si necesitas upscaling/refiner).

2) Modelo de video/realtime para animación.

Text → Subtitle

Usa un modelo de texto para generar y sincronizar subtítulos desde un guion.

Text → Music

1) Modelo de texto para guiar el estilo.

2) Motor de audio (TTS/Music) para generar audio final.

Idea → Reel Carrusel

1) Modelo de texto estructura las diapositivas.

2) Modelo de imagen crea los visuales.
